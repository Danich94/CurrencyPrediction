# CurrencyPrediction
Предсказание курса доллара с использованием временных рядов.
### Описание проекта
На основе котировок курса доллара по отношению к рублю попробуем построить прогноз на каждый день на месяц вперед от конца данных. Для прогнозирования курса доллара будем использовать прогнозирующую модель ARIMA из библиотеки statsmodels.
### Исходные данные
Для построения прогнозирующей модели воспользуемся данными о валютной паре 'доллар/рубль' с сайта компании Финам (https://finam.ru), представляющими собой цену доллара в рублях на момент закрытия биржевой сессии за период времени выбранный пользователем.
